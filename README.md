# JHP Coding: Stats and ML Study Kit

## 소개

`JHP_coding`은 데이터마이닝, 통계학, 머신러닝 수학을 학습하면서 핵심 개념을 직접 구현하고 정리하는 Python 기반 학습 리포지터리입니다.

이 프로젝트의 목적은 단순히 라이브러리를 사용하는 데 그치지 않고, 회귀분석, 잔차 분석, 평가 지표, 경사하강법, 클러스터링 등 데이터 분석의 기본 원리를 코드로 확인하는 것입니다.

산업공학 전공 학부생의 관점에서 수업과 개인 학습에서 다루는 개념을 재현 가능한 예제와 함께 정리하는 것을 목표로 합니다.

## 주요 기능

- 단순선형회귀 직접 구현
- 평균제곱오차, 평균절대오차, 결정계수 등 기본 평가 지표 구현
- 회귀 예제 실행 코드 제공
- 데이터마이닝 및 머신러닝 수학 예제 확장 예정
- 학습 내용을 코드와 문서로 정리하는 오픈소스 형식의 저장소 운영
